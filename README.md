# 👟 SneakHub - E-Commerce Website

SneakHub is a modern and responsive e-commerce website built using **HTML, Tailwind CSS, and JavaScript**. The project provides a smooth shopping experience with interactive features such as a shopping cart, wishlist, page navigation, animations, and responsive design.

## 🚀 Features

- 🛍️ Responsive E-Commerce UI
- 📱 Mobile-Friendly Navigation
- ⚡ Single Page Application (SPA) Navigation
- 🛒 Shopping Cart Functionality
- 💾 Cart Data Stored using LocalStorage
- ❤️ Wishlist Toggle
- 🔔 Toast Notifications
- ✨ Smooth Animations & Scroll Reveal Effects
- 📩 Contact Form
- 📧 Newsletter Subscription Form
- 🎨 Modern UI with Tailwind CSS

## 🛠️ Technologies Used

- HTML5
- Tailwind CSS
- JavaScript (ES6)

## 💡 JavaScript Concepts Used

- DOM Manipulation

- Template Literals
